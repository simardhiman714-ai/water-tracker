#  💧 Water Tracker App
A modern, interactive and beginner-friendly web app to track daily water intake and build healthy hydration habits 🚰
---------------------
## ✨ Overview
Water Tracker is a simple yet powerful mini project that helps users monitor how much water they drink in a day. It encourages healthy hydration habits with a clean UI and smooth interaction.
---------------------
##🚀 Features
➕ Add water intake easily (glass/button based system)
📊 Real-time progress tracking
🎯 Daily goal monitoring
🔄 Reset option for new day
🎵 Optional sound effects (if added in project)
💻 Responsive and clean UI
---------------------
##🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & UI design
JavaScript – Functionality & logic
